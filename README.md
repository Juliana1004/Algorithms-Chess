# Project-chess

Entrega de medio término del proyecto Chess hecho en Java, se hace el uso de 2 algoritmos de ordenamiento (Bubble Sort e Insertion Sort) y arrays randoms con el fin de hacer una mejor prueba de los algoritmos.


## Como iniciar

Ejecuta el programa por consola de la siguiente forma
java Main a=ALGORITMO c=TIPO o=COLOR

* Valores válidos:
* a: b(Bubble Sort) o i(Insertion Sort)
* t: c(Carácter) o i(Entero)
* o: b(Blancas) o n(Negras)

Ej: a=b c=i o=n

## Claes Principales

* Main: Encargada de empezar la ejecución del programa de ordenamiento y creación de arrays.
* Validations: Encargada de validar los datos ingresados sean correctos.
* Bubble Sort e Insertion Sort: Implementan los algoritmos de ordenamiento.
